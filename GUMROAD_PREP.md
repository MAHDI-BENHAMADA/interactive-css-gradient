# Interactive Gradient – Gumroad Product Package

## 📦 Package Contents

### For Customers Who Want Quick Results (No Coding)

**Location:** `examples/` folder

- **index.html** – Landing page with links to all demos
- **dark-theme.html** – Pre-built dark theme (standalone, copy & use)
- **vibrant-colors.html** – Pre-built vibrant theme (standalone, copy & use)
- **minimalist.html** – Pre-built minimalist theme (standalone, copy & use)

**How customers use it:**
1. Open `examples/index.html` in a browser to see all demos
2. Right-click → "Save As" on any example to download
3. Or copy the HTML code into their project
4. Edit colors/speeds directly in the HTML `<style>` section

### For Developers (Full Source Code)

**Location:** `src/` folder + configuration files

- **src/main.ts** – TypeScript mouse-tracking logic
- **src/style.scss** – SCSS styling with animations
- **src/assets/** – Static assets folder
- **index.html** – Main entry point
- **package.json** – Dependencies and build scripts
- **tsconfig.json** – TypeScript configuration

**How developers use it:**
1. Run `npm install`
2. Run `npm run dev` for development
3. Edit `src/style.scss` and `src/main.ts`
4. Run `npm run build` for production

### Pre-Built Production Files

**Location:** `dist/` folder

- **dist/index.html** – Compiled HTML
- **dist/assets/index-[hash].css** – Minified CSS
- **dist/assets/index-[hash].js** – Minified JavaScript

These are ready to deploy directly (no build needed).

## 📖 Documentation Files

- **README.md** – Main product page & feature overview
- **CUSTOMIZATION.md** – Detailed customization guide with code examples
- **LICENSE** – MIT License (free to use commercially)

## 🚀 Customer Onboarding
